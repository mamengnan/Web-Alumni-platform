##  To get started:

     1: 下载代码库, https://git.coding.net/Blust/Web-Alumni-platform.git
     
     2: 前端开发无需特定IDE，文本编辑器即可。
     
     3: 在本机配置好Node.js开发环境(node,npm)。
     
     4: 配置完成后直接进入项目，在终端输入 cnpm install(cnpm 配置自行百度) 或者 npm install。
     
     5: 以上配置完成后，在项目内直接运行 npm start。
